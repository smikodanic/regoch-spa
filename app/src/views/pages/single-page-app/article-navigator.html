navigator

instance of the class Navigator

A library for navigate through the URLs by changing browser's address bar data.

Properties

There is no properties in the class.

Methods

Use this methods in the controller.

  • goto (url :string, state :string, title :string): void

    Navigates to a view using an absolute URL path. More at https://developer.mozilla.org/en-US/docs/Web/API/History/pushState
    url - absolute URL path, for example: '/customer/product/25'
    state - The state data. Fetch it with event.detail.
    title - Not used in major browsers.

  • forward (): void

    Go forward like forward button is clicked.

  • back (): void

    Go back like back button is clicked.

  • go (delta :number): void

    Loads a specific page from the session history. You can use it to move forwards and backwards through the history depending on the delta value.
    delta - history index number, for example: -1 is like back(), and 1 is like forward()

  • reload (): void

    Reloads the page like refresh button is clicked.



    • LISTENERS

  • onPushstate (listener :Function): void

    Listen for the 'pushstate' event, for example when data-rg-href is clicked or when goto() is used.
    listener - callback function which will be executed on 'pushstate' event

  • onPopstate (listener :Function): void

    Listen for the 'popstate' event, for example when BACK,FORWARD button is clicked or when back(),forward() is used.
    listener - callback function which will be executed on 'popstate' event

Example


const { Controller, syslib } = require('regoch-spa');

class MyCtrl extends Controller {
  async redirectMe() {
    syslib.navigator.goto('/admin/products/list');
  }
}
