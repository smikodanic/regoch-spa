util

instance of the class Util

A library of utilities and helpers.

Methods

Use this methods in the controller.

  • async sleep (ms:number): Promise<void>

    make some time delay in miliseconds

Example


class MyCtrl extends Controller {
  constructor(app) {
    this.util = app.sys.util;
  }

  async test() {
    await this.util.sleep(1300); // wait for 1.3 seconds before sending HTTP request
    const answer = await this.hc.askJSON('https://jsonplaceholder.typicode.com/todos/1');
  }
}
