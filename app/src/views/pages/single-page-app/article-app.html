App

class App

Properties

Property Description Type Default
CONF Configuration injected from the /conf/ files. Use it in the controller. any {}
CONST Constants. Use it in the controller. any {}
sys Injected system libraries which can be used in the controller: eventEmitter, util, Form, HTTPClient, Cookie object {}
lib Injected custom made libraries. object {}
controllers A collection of all controllers. Make it possible to use a controller's methods inside another controller. object {}

Methods

  • conf (name:string, value:any): App - inject configuration which can be used in the controller
  • const (name:string, value:any): App - inject constant which can be used in the controller
  • freeze (): void - freeze the CONF and CONST to prevent unwanted modifications in the controller
  • libInject (libs:object): void - inject lib which can be used in the controller
  • libEmpty (): void - empty the lib property
  • controller (Ctrls:class): void - create controller instances and inject into the this.controllers
  • routes (routesCnf:string[][]): App - define the app routes
  • run (): void - run the app, i.e. execute the router

Example


const { App } = require('regoch-spa');

// conf
const apiCnf = {baseURL: 'https://jsonplaceholder.typicode.com'};

// lib
const StringExt = require('./lib/StringExt');
const Rand = require('./lib/Rand');

// controllers
const HomeCtrl = require('./controllers/HomeCtrl');
const SinglePageAppCtrl = require('./controllers/SinglePageAppCtrl');
const NotfoundCtrl = require('./controllers/NotfoundCtrl');



const app = new App();

app
  .conf('API', apiCnf)
  .const('myNum', 10)
  .const('myStr', 'some thing')
  .const('myObj', {a: 22})
  .freeze();

app.libInject({ StringExt, Rand });

app
  .controller([HomeCtrl, SinglePageAppCtrl, NotfoundCtrl])
  .routes(routesCnf)
  .run();
