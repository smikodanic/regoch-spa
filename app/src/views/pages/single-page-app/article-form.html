Form

class Form

Get and set the HTML form field values.

Properties

Property Description Type Default
formName the form name used in data-rg-form string

Methods

Use this methods in the controller after "Cookie" class is instantiated.

  • setControl (key:string, val:string): <void>

    Set the form control value.
    key - the value of the "name" HTML attribute
    val - the value

  • getControl (key:string): <string|number>

    Get the form control value. Returned value is string.
    key - the value of the "name" HTML attribute

  • delControl (key:string): <void>

    Empty the form control value.
    key - the value of the "name" HTML attribute

Example


class MyCtrl extends Controller {
  constructor(app) {
    this.myF = new app.sys.Form('myForm');
  }

  test() {
    const firstName = this.myF.getControl('first_name');
    console.log(firstName)
  }

}
  

<form data-rg-form="myForm">
  <input type="text" name="first_name">
</form>
