Transitional Variable

The transitional variable is the argument in the router middleware functions (trx) => { ... }. In order to function properly the minimal and requred property is uri. Sometimes body property will be also required.
Example: trx = {uri: '/product/save', body: {name: 'Toy', color: 'red'}}

Methods

Use this methods to control the websocket server routes.

  • constructor (routerOpts :object) :void

    Create the Router class instance.

  • set trx(obj :{uri, body, ...}) :void

    Set the transitional variable.

  • get trx() :{uri, body, ...}

    Get the transitional variable.

  • def (route :string, ...funcs :Function[]) :Router

    Define route, routeParsed and associate corresponding functions to the route.

  • redirect (fromRoute :string, toRoute :string) :Router

    Redirect from one route to another route.

  • notfound (...funcs :Function[]) :Router

    Execute funcs when no route found.

  • do (...funcs :Function[]) :Router

    Execute funcs always on every route.

  • exe () :trx

    Execute the defined routes.
